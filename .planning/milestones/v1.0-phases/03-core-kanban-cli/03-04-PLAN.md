**3. Create apps/cli/src/commands/column/reorder.ts (COLM-03, D-13):**
Export `columnReorderCommand(id: string, options: { position: string }, globalOptions: GlobalOptions): Promise<void>`

- Parse position: `const targetPos = parseInt(options.position, 10)`. Validate it's a positive integer, error if not.
- Fetch all columns for the board: need to find the column being moved first to get its boardId.
  1. GET the column: `apiRequest(`/api/v1/columns/${id}`)` — response gives `data.boardId`
  2. GET all columns: `apiRequest(`/api/v1/boards/${data.boardId}/columns?limit=100`)` — sorted by position
- Remove the target column from the list to get `remainingColumns`
- Validate `targetPos` is between 1 and `remainingColumns.length + 1` (inclusive). Error if out of range: `"Position must be between 1 and {max}"`
- Calculate fractional index using `generateKeyBetween` from `fractional-indexing`:
  ```typescript
  import { generateKeyBetween } from 'fractional-indexing'

  let newPosition: string
  if (targetPos === 1) {
    // Before first remaining column
    newPosition = generateKeyBetween(null, remainingColumns[0]?.position ?? null)
  } else if (targetPos > remainingColumns.length) {
    // After last remaining column
    newPosition = generateKeyBetween(remainingColumns[remainingColumns.length - 1].position, null)
  } else {
    // Between two columns
    newPosition = generateKeyBetween(
      remainingColumns[targetPos - 2].position,
      remainingColumns[targetPos - 1].position
    )
  }
  ```
- PATCH: `apiRequest(`/api/v1/columns/${id}`, { method: 'PATCH', body: JSON.stringify({ position: newPosition }) })`
- Success per UI-SPEC: `Moved column {data.name} to position {targetPos}`
- 3-mode output

**1. Create apps/cli/src/commands/column/delete.ts (COLM-04, D-12):**
Export `columnDeleteCommand(id: string, options: { force?: boolean; moveTo?: string }, globalOptions: GlobalOptions): Promise<void>`

Implementation flow per D-12 and UI-SPEC:

```
Step 1: GET the column to validate existence and get name
  apiRequest(`/api/v1/columns/${id}`)
  If 404: show "Column not found..." error and exit

Step 2: GET items in column to check if non-empty
  apiRequest(`/api/v1/columns/${id}/items?limit=1`)
  Parse response. Check data.length or meta.total to see if items exist.

Step 3: Handle items logic per D-12
  IF column has items:
    IF options.moveTo is set:
      - Fetch ALL items: apiRequest(`/api/v1/columns/${id}/items?limit=200`)
      - Move each item to target column using Promise.all with concurrency limit:
        Batch items into groups of 5, for each batch:
          await Promise.all(batch.map(item =>
            apiRequest(`/api/v1/items/${item.id}/move`, {
              method: 'POST',
              body: JSON.stringify({ columnId: options.moveTo })
            })
          ))
      - After all items moved, proceed to delete
      - Success: "Moved {N} items to column {targetId} and deleted column {name} ({id})"
    ELSE IF options.force:
      - Confirm with user: "Delete column "{name}" and its {N} items? [y/N]"
      - If declined, exit
      - If confirmed, proceed to delete (API cascade handles item deletion)
      - Success: "Deleted column {name} ({id})"
    ELSE (no --move-to, no --force):
      - Error per UI-SPEC: 'Column "{name}" has {N} items. Use --move-to <column-id> to relocate them, or --force to delete them.'
      - Set process.exitCode = 1 and return

  IF column is empty (no items):
    - No confirmation needed (per D-12: empty columns delete without prompts)
    - Proceed directly to delete

Step 4: Delete the column
  apiRequest(`/api/v1/columns/${id}`, { method: 'DELETE' })
  204 response

Step 5: Output
  - JSON: outputJson({ deleted: true, id, itemsMoved: N (if --move-to) }, {})
  - Ink: SuccessMessage with appropriate message
  - Plain: outputPlain with same message
```

For the concurrency-limited move, implement a simple helper:
```typescript
async function moveItemsInBatches(
  items: ItemResponse[],
  targetColumnId: string,
  batchSize = 5
): Promise<void> {
  for (let i = 0; i < items.length; i += batchSize) {
    const batch = items.slice(i, i + batchSize)
    await Promise.all(
      batch.map((item) =>
        apiRequest(`/api/v1/items/${item.id}/move`, {
          method: 'POST',
          body: JSON.stringify({ columnId: targetColumnId }),
        })
      )
    )
  }
}
```

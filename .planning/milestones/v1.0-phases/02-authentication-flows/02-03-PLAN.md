**2. Logout command (`apps/cli/src/commands/auth/logout.ts`) — per D-12, AUTH-03:**

Export `async function logoutCommand(globalOptions: GlobalOptions): Promise<void>`

Logic:
1. Check `hasTokens()`. If no tokens:
   - `json` mode: `outputJson({ message: 'Not currently logged in' }, {})`
   - `ink` mode: print dim text "Not currently logged in."
   - `plain` mode: print "Not currently logged in."
   - Return.
2. Read refresh token from config: `config.get('refreshToken')`
3. Attempt API revoke (best effort, per D-12): `fetch(${apiBaseUrl}/api/v1/auth/logout, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ refreshToken }) })`. Wrap in try/catch, ignore errors.
4. Call `clearTokens()` to remove local tokens.
5. Output success:
   - `json` mode: `outputJson({ message: 'Logged out successfully' }, {})`
   - `ink` mode: print green bold "Logged out successfully."
   - `plain` mode: print "Logged out successfully."

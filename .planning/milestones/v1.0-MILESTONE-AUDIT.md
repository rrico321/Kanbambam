### Integration Checker Notes

The integration checker initially flagged a "critical" issue with web auth (cookie vs Bearer header). **This was a false positive** — the Hono auth middleware at `apps/api/src/middleware/auth.ts:4` explicitly configures `cookie: 'access_token'`, accepting JWT from both Authorization header and the httpOnly cookie.

**Low-severity finding:** CLI does not import `@kanbambam/shared` despite it being a declared dependency. Hand-written types in `apps/cli/src/types.ts` duplicate the shared Zod schemas. This creates a type drift risk but causes no runtime issues today.
